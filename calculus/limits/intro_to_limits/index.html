Meaning and Notation

The limit of a function represents the value which a function approaches, but never reaches. Consider this graph of f(x). Even though f(2) isn't defined, a glance at the graph tells you that f(x) appears to approach 1 as x gets infinitely close to—or approaches—2.

The following equation demonstrates the notation of your average limit. It would be read aloud as "the limit of f(x) as x approaches c is L."

\lim\limits_{x \to c} f(x) = L

The arrow underneath the limit operator (\to) simply means "approaches." The limit of the aforementioned graph, as x approaches 2, would be written like so:

\lim\limits_{x \to 2} \frac{x(x-2)}{2(x-2)} = 1

Always remember! The limit of a function represents a value that is infinitely close to—but never reaching—another value. That is why \lim\limits_{x \to 2} f(x) can be 1 while f(2) is \undef.

Limit Examples

\lim\limits_{x \to 2} g(x) = 2 while g(2) = \undef because, similar to the previous example, the graph converges on y=2 as x draws ever closer to 2.

\lim\limits_{x \to 0} g(x) = 0 even when g(0) = 0 as well, because limits still function the same even when f(c) is defined.

\lim\limits_{x \to 1} g(x) = \frac{1}{2} despite the fact that g(1) = 1 because the limit of a function is what it approaches, not what it actually is at f(c).

Limits With Infinity

\lim\limits_{x \to 0} h(x) = \infin because both branches grow faster and faster as x nears 0. You're allowed to write that the limit equals infinity because, being a limit, it never truly does equal infinity.

\lim\limits_{x \to \infin} h(x) = 0 and \lim\limits_{x \to -\infin} h(x) = 0 because the graph is always getting closer to zero as x grows larger in either direction. Limits to positive or negative infinity are the same thing as end behavior.

You may have noticed that a limit at one of the asymtotes, \lim\limits_{x \to -1} h(x), was never solved for. One branch goes up, while the other goes down—they never converge on a matching value. What does this mean for the limit? That issue will be covered next.

Summary

  • This is a limit: \lim\limits_{x \to c} f(x) = L
  • As x approaches c, f(x) approaches L.
  • Limits represent what a function approaches, but never reaches.
