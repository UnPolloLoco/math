Direct Substitution

In friendly situations where c (in x \to c) is defined and continuous, directly substituting c into the limit's function would work just fine. Polynomials will always allow direct substitution, as they have no discontinuity.

\begin{aligned} \lim\limits_{x \to -2 } \big(x^3 - 2x \big) \eq ( -2 )^3 - 2( -2 ) \n \eq -8 + 4 \n \eq -4 \end{aligned}
\lim\limits_{x \to -2 } \big(x^3 - 2x \big) = 4

Factoring

If direct substitution results in an indeterminate form such as \frac{0}{0}, then factoring may help. This method works when c is a holeâ€”that is, when all (x-c) factors in the denominator can be canceled.
The additional condition of x \neq 4 for \frac{1}{x+4} (which comes from the factors being canceled out) can be ignored in this context.

\begin{aligned} \lim\limits_{x \to 4} \bigg( \frac{x-4}{x^2 - 16} \bigg) \eq \coldim{ \frac{(4)-4}{(4)^2 - 16} = \frac{0}{0} } \nn \lim\limits_{x \to 4} \bigg( \frac{x-4}{ (x-4)(x+4) } \bigg) \eq \nn \lim\limits_{x \to 4} \bigg( \frac{ \cancel{ x-4}}{ \cancel{ (x-4)} (x+4)} \bigg) \eq \nn \lim\limits_{x \to 4} \bigg( \frac{1}{x+4} \bigg) \eq \frac{1}{(4)+4} \nn \eq \frac{1}{8} \end{aligned}
\lim\limits_{x \to 4} \bigg( \dfrac{x-4}{x^2 - 16} \bigg) = \dfrac{1}{8}

Rationalization

Even though rationalization is more commonly used to take radicals out of the denominator, it can also be used to solve some limits too. To rationalize, multiply both sides of the fraction by the conjugate of whichever side contains the radical.

\begin{aligned} \lim\limits_{x \to 2} \Bigg( \frac{\sqrt{2x} - 2}{x - 2} \Bigg) \eq \coldim{ \frac{\sqrt{2(2)} - 2}{(2) - 2} = \frac{0}{0} } \n \lim\limits_{x \to 2} \Bigg( \frac{\sqrt{2x} - 2}{x - 2} \cdot \frac{\sqrt{2x} + 2}{\sqrt{2x} + 2} \Bigg) \eq \n \lim\limits_{x \to 2} \Bigg( \frac{2x - 4}{(x-2)(\sqrt{2x} + 2)} \Bigg) \eq \n \lim\limits_{x \to 2} \Bigg( \frac{2(x - 2)}{(x-2)(\sqrt{2x} + 2)} \Bigg) \eq \n \lim\limits_{x \to 2} \Bigg( \frac{2 \cancel{(x - 2)}}{\cancel{(x-2)}(\sqrt{2x} + 2)} \Bigg) \eq \n \lim\limits_{x \to 2} \Bigg( \frac{2}{\sqrt{2x} + 2} \Bigg) \eq \frac{2}{\sqrt{2(2)} + 2} \n \eq \frac{1}{2} \end{aligned}
\lim\limits_{x \to 2} \Bigg( \dfrac{\sqrt{2x} - 2}{x - 2} \Bigg) = \dfrac{1}{2}

Summary

  • Directly substituting c into f(x) works with common, friendly functions
  • If direct substitution results in an indeterminate form, then the function must be modified in order to solve
  • Strategy 1: Factor both sides of the fraction and simplify
  • Strategy 2: Rationalize and simplify
