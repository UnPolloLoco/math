Undefined vs. Indeterminate

If a value is undefined, then it has no value. If a value is indeterminate, then the value exists but is useless. Consider these two common headaches, \frac{n}{0} and \frac{0}{0}.

3 \div 0, for example, is \undef because it doesn't have an answer. After rearranging the fraction into a multiplication problem, it's clear that not even infinity can bring 0x to 3.

\begin{aligned} 3 \div 0 &= x \n 3 &= 0x \end{aligned}

0 \div 0 is \ind because it has infinite answers, and therefore tells you nothing. Of course, any number multiplied by zero will be zero.

\begin{aligned} 0 \div 0 &= x \n 0 &= 0x \end{aligned}

Indeterminate Forms

An indeterminate form is a value which carries no meaning on its own, and does not relate to the actual answer; it's like asking a limit for a number, but recieving an exquisite painting of an apple instead. If direct substitution results in an indeterminate form (e.g. \lim\limits_{x \to 4} \frac{x - 4}{x^2 - 16} = \frac{(4) - 4}{(4)^2 - 16} = \frac{0}{0}), then the function has to be modified in order to solve the limit (e.g. \lim\limits_{x \to 4} \frac{x - 4}{x^2 - 16} = \lim\limits_{x \to 4} \frac{1}{x+4} = \frac{1}{8}). The seven indeterminate forms are as follows:

\frac{0}{0}, \quad 0^0, \quad \infin - \infin, \quad \frac{\infin}{\infin}, \n 0(\infin), \quad 1^{\infin}, \quad {\infin}^0

All of those values have a reason for being indeterminate:

\frac{0}{0}

As previously discussed, \frac{0}{0} is indeterminate because any value of x will satisfy its rearranged version 0 = 0x. There is no way to go off of \frac{0}{0} to get any helpful answer.

0^0

This problem originates from conflicting exponent properties a^0 = 1 and 0^a = 0. When limits are used to attempt to find a common ground, it quickly becomes apparent that 0^0 has several possibilities: \lim\limits_{a \to 0^+} a^0 and \lim\limits_{a \to 0^+} 0^a both directly substitute to 0^0, but \lim\limits_{a \to 0^+} a^0 = 1 while \lim\limits_{a \to 0^+} 0^a = 0. Since 0^0 cannot be both 0 and 1, it's left as indeterminate.
Keep in mind that these are indeterminate forms of limitsâ€”it isn't a true 0^0, but rather something approaching 0 to the power of something approaching 0. Outside of limits, 0^0 is often considered to be 1.

\infin - \infin

Seeing as infinity a concept rather than a number, operating on it isn't as clean as it is with actual numbers. It's not equal to 0 because there's no certainty that each infinity is equivalent. The answer could be anything. An example of this would be \lim\limits_{x \to \infin} (2x - x).

\frac{\infin}{\infin}

This fraction has the same issue that \infin - \infin has. You can't tell how much larger one infinity is than the other, so \frac{\infin}{\infin} is pretty much worthless.

0(\infin)

At first, this one seems nonsensical. How could something multiplied by zero not just be zero? Consider these similar limits, \lim\limits_{x \to 0} (x \cdot \frac{1}{x}) = 1 and \lim\limits_{x \to 0} (x \cdot \frac{5}{x}) = 5. They both directly substitute to 0(\infin), but have different values.

1^{\infin}

Both \lim\limits_{x \to \infin} (1 + \frac{1}{x})^x and \lim\limits_{x \to \infin} (1 + \frac{1}{x})^{x^2} substitute to 1^{\infin}, but they equal e and \infin respectively. This discrepancy is caused by whether or not 1 is approached faster than \infin is.

{\infin}^0

Limits \lim\limits_{x \to \infin} x^{\frac{1}{x}} = 1 and \lim\limits_{x \to \infin} \big(5^x\big)^{\frac{1}{x}} = 5 will both substitute to {\infin}^0 even though they equal different things.

Summary

  • \undef means no answers
  • \ind means you can't figure out the answer, as what you are given can mean many things
  • If direct substitution of a limit leads to an indeterminate form, then the limit's function must be modified before attempting to solve
